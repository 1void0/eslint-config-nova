# @void0/eslint-config

This package provides a strict stylelint config for projects using Prettier.

## Usage

1. Install the package:

```sh
npm install --save-dev @void0/stylelint-config
```

2. Add `"extends": "@void0/stylelint-config"` to your .stylelintrc.
